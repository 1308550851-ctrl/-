## 1. 项目概述

### 1.1 项目背景
本项目是一个电竞主题静态网页设计任务，要求使用AI辅助开发工具完成。根据考试要求，需要创建一个视觉效果出色、交互体验良好的电竞主题网站。

### 1.2 AI工具使用策略
- 使用AI进行需求分析和PRD编写
- 使用AI生成HTML/CSS/JavaScript代码
- 使用AI进行视觉设计建议
- 使用AI进行代码优化和调试

---

## 2. Prompt设计思路

### 2.1 Prompt设计原则

1. **明确性原则**：Prompt应清晰明确，避免歧义
2. **结构化原则**：采用分点式、层次化的Prompt结构
3. **示例驱动**：提供参考示例帮助AI理解期望输出
4. **迭代优化**：根据AI输出不断优化Prompt

### 2.2 Prompt分类

| 类型 | 用途 | 示例 |
| :--- | :--- | :--- |
| 需求分析 | 理解用户需求 | "分析这份考试题目，帮我理解要求" |
| 设计建议 | 获取设计灵感 | "推荐电竞风格的配色方案" |
| 代码生成 | 生成代码 | "创建一个电竞风格的导航栏" |
| 代码优化 | 改进现有代码 | "优化这段CSS动画性能" |
| 问题调试 | 解决bug | "修复JavaScript报错" |

## 3. 核心Prompt记录

### 3.1 需求分析Prompt

```
你是一位资深的产品经理和UI设计师。请帮我分析以下考试题目要求：

考试题目：
- 基于Vibe Coding（自然语言编程）模式
- 利用AI辅助开发工具设计一款与电竞场景相关的静态视觉网页产品
- 产品需要具备较高的审美水准、创新度及实用价值

评分标准：
1. AIGC基本素养（40分）：需求拆解、PRD文档、Prompt文档、使用场景清晰
2. AIGC视觉效果（30分）：页面元素设计、排版布局、视觉美感
3. 创新度（30分）：创新思维、独特设计

请帮我分析：
1. 核心需求点
2. 页面结构建议
3. 视觉风格方向
4. 功能模块规划
```

**AI输出摘要**：
- 核心需求：创建电竞主题静态网页，展示热门游戏、赛事、战队等内容
- 页面结构：首页、游戏、赛事、战队、资讯、页脚
- 视觉风格：深色科技风、霓虹渐变、粒子动画
- 功能模块：导航、英雄区、游戏展示、赛事展示、战队展示、新闻资讯

---

### 3.2 设计风格Prompt

```
我需要创建一个电竞主题网站，请推荐：

1. 配色方案（主色、辅助色、背景色）
2. 字体选择（标题字体、正文字体）
3. 视觉效果（动画、渐变、阴影）
4. 设计风格参考

要求：
- 科技感强
- 视觉冲击力大
- 符合电竞文化
```

**AI输出摘要**：
- 配色：主色#00f5ff（青色），辅助色#ff00ff（粉色）、#7b00ff（紫色）
- 字体：Orbitron（标题）、Rajdhani（正文）
- 效果：霓虹发光、粒子背景、渐变过渡
- 参考：赛博朋克风格、游戏UI设计

---

### 3.3 HTML结构Prompt

```
请帮我创建一个电竞主题网站的HTML结构，包含：

1. 导航栏（Logo、导航链接、登录/注册按钮）
2. 英雄区（大标题、副标题、统计数据、CTA按钮）

4. 赛事展示区（主赛事+侧边赛事列表）
5. 战队展示区（4支顶级战队卡片）
6. 新闻资讯区（主新闻+侧边新闻列表）
7. 页脚（品牌信息、链接、邮件订阅）

要求：
- 使用语义化标签
- 添加合适的class命名
- 结构清晰
```

**AI输出摘要**：生成了完整的HTML结构，包含所有要求的模块，使用了语义化标签如`<nav>`, `<section>`, `<footer>`等。

## 6. AI使用经验总结

### 6.1 成功经验

1. **清晰的Prompt结构**：使用分点式、层次化的Prompt更容易获得准确结果
2. **提供参考示例**：给出参考链接或示例代码帮助AI理解期望输出
3. **逐步细化需求**：从整体到细节逐步提出要求
4. **及时验证反馈**：对AI输出进行验证并及时反馈调整

### 6.2 改进方向

1. **更精确的需求描述**：减少歧义，提高Prompt质量
2. **代码规范要求**：明确代码风格和命名规范
3. **性能要求明确**：在Prompt中明确性能指标
